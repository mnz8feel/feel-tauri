# Tauri 官方插件功能速览（References）

本文基于 Tauri v2 文档 `reference/javascript` 中的官方插件列表整理，面向快速选型。

| 插件                | 主要功能                                    | 常见场景                        |
| ------------------- | ------------------------------------------- | ------------------------------- |
| `autostart`         | 管理应用开机自启动                          | 桌面常驻工具、启动即同步        |
| `barcode-scanner`   | 条码/二维码扫描能力                         | 资产盘点、出入库、移动扫码      |
| `biometric`         | 生物识别认证（指纹/面容）                   | 二次确认、敏感操作授权          |
| `cli`               | 解析命令行参数                              | 支持 `--mode`、批处理、脚本调用 |
| `clipboard-manager` | 读写系统剪贴板                              | 复制增强、粘贴板工具            |
| `deep-link`         | 注册和处理深链接                            | OAuth 回跳、从浏览器唤起应用    |
| `dialog`            | 原生系统对话框（打开/保存/确认）            | 文件选择、确认弹窗              |
| `fs`                | 文件系统访问（受权限控制）                  | 读写配置、导入导出文件          |
| `geolocation`       | 获取地理位置                                | 地理围栏、定位展示              |
| `global-shortcut`   | 全局快捷键                                  | 全局唤醒、快捷操作              |
| `haptics`           | 触觉反馈（振动）                            | 移动端轻反馈、交互确认          |
| `http`              | 原生 HTTP 请求能力                          | 跨域受限场景、统一网络层        |
| `log`               | 结构化日志输出                              | 诊断、线上问题排查              |
| `nfc`               | NFC 读写/交互能力                           | 门禁卡、标签读取、配对          |
| `notification`      | 系统通知                                    | 消息提醒、后台任务通知          |
| `opener`            | 使用系统默认程序打开资源                    | 打开外链、文件、文件夹          |
| `os`                | 获取 OS/平台相关信息                        | 平台分支逻辑、环境判断          |
| `positioner`        | 窗口定位与贴边策略                          | 托盘弹窗、跟随窗口定位          |
| `process`           | 进程相关能力（重启/退出等）                 | 应用重启、生命周期控制          |
| `shell`             | 执行外部命令/打开 shell 资源（受权限控制）  | 调用系统工具、集成外部程序      |
| `sql`               | 内置数据库访问封装（SQLite/MySQL/Postgres） | 本地缓存、离线数据、业务存储    |
| `store`             | 键值存储（本地持久化）                      | 用户设置、轻量状态保存          |
| `stronghold`        | 基于 Stronghold 的安全存储                  | 密钥、敏感凭据安全保存          |
| `updater`           | 应用更新检查与安装                          | 自动更新、灰度发布              |
| `upload`            | 文件上传辅助能力                            | 大文件上传、分片上传流程        |
| `websocket`         | WebSocket 客户端通信                        | 实时消息、推送、长连接          |
| `window-state`      | 记忆窗口状态（大小/位置/最大化）            | 提升桌面应用体验                |

## 使用建议

- 权限最小化：先只开必需权限，再逐步扩展。
- 插件优先：能用官方插件就优先用，跨平台行为更稳定。
- 分平台能力：如 `nfc`、`haptics`、`geolocation`，先确认目标平台支持。
- 安全相关：`shell`、`fs`、`deep-link` 建议重点审查 ACL 与输入来源。
